*** Guide-to-Links ***
 EL

EL connects certain words like "someone/everyone/no_one/anyone", "somebody (etc.)", "somewhere(etc.)", "who/what/where(etc.)" and "whatever/whoever(etc.)" to the word "else". 

         +---Ss*w--+
         +-EL--+   |
         |     |   |
        What else is new
Curiously, assorted idiomatic profanity can take exactly the same places as "else" can; these also take an EL- connector. 
         +-----EL------+  
         |   +_IOG+_IOF+ 
         |   |    |    | 
       what in   the hell happened ?
