 QN

QN connects verbs to question-words, in an inverted fashion: 

          +-----QN------+
          |             |
    It   happend       when?
    You  hit           how     many home runs?
    You  want          which   one?
    You  know this     how?
    You  hit the car   how?
Almost all verbs can be used in this inverted fashion.

This link has some similarities to the QI link used for indirect questions; it should not be confused with it.
